How to use our API system

Dorl Neylon

SimpleCode's API system's JSON-based default-type API.


Step 1. Introduction.

To create a post you should go to /createpost with author, title and content fields. For example,

$.post("/createpost", { "author" : "Dorl Neylon", "title" : "How to use our API system", "content" : "Some stuff here" })

As an output you'll get something like this:

{ "link" : "/token", "author" : "testauthor", "title" : "testtitle", "content" : "testcontent", "publish date" : "01.01.2000" }

Token is a 5-letters code situated in the link. You'll need a token if you want to check other post's info. To do that, go to /checkpost page with the link field. For example,

$.post("/checkpost", { "link" : "qewcz" })

As an output you'll get the same thing you could get from the /createpost page.


Step 2. Formats?

There're ways to post not only ridiculously simple posts. Below these there are codewords for formatting.

  • [NEWLINE] - this means a new line starts. Please, don't use too long lines.
  • [ENDLINE] - this means the end of the line.
  • [EMPTYLINE] - this will be replaced with a <br> tag.
  • [BOLD] - this will start your bold text.
  • [/BOLD] - this'll end your bold text.
  • [ITALIC] - this'll start your italic text.
  • [/ITALIC] - this'll end your italic text.
  • [QUOTE] - this'll start your quote. Please, don't use too long quotes.
  • [/QUOTE] - this'll end your quote.
  • [H2] - this'll remake your text into a header.
  • [/H2] - the end of the header.
  • [H3] - the start of the sub-header.
  • [/H3] - the end of the sub-header.
  • [CODE] - the start of the code block.
  • [/CODE] - the end of the code block.
  • [TABLE] - the start of the table.
  • [/TABLE] - the end of the table.
  • [TR] - the start of the new row in a table. Replaces with <tr> tag.
  • [/TR] - the end the row in a table.
  • [TD] - the start of the new block in a row in a table. Replaces with <td> tag.
  • [/TD] - the end of the block in a row in a table.

For example,

let content = "[NEWLINE]Plain text[ENDLINE][NEWLINE] \
               [BOLD]Bold and[/BOLD][ITALIC]italic[/ITALIC] text[ENDLINE] \
               [CODE][NEWLINE]welcome = lambda d: print(f'hello, {d}')[ENDLINE] \
               [NEWLINE]welcome('Dorl Neylon')[ENDLINE][/CODE] \
               [TABLE][TR][TD]abc[/TD][TD]def[/TD][/TR] \
               [TR][TD]asdf[/TD][TD]zxcv[/TD][/TR][/TABLE]".split('               ').join('');
$.post("/createpost", { "author" : "Dorl Neylon", "title" : "How to use our API system", "content" : content })
